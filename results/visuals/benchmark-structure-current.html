Verdict: OURS vượt chuẩn ở multi-type scope · statistical rigor · tách 2 lớp metric · leakage check (cả 2 paper đều thiếu). Ngang registry scale + split protocol. Thiếu: Neo4j 27 DB (13% registry, directional) · density q/DB thấp hơn (bù DB-macro + bootstrap) · chưa public release. Gap κ → đóng bằng quyết định bỏ T1/T2 (GT inherit → κ N/A, đúng như chính 2 paper này).

Đường khác biệt vs SOTA: biểu diễn (card vs raw DDL) · tầng lọc domain (nén pool) · agent tie-break ở quyết định · mở rộng SQL-only → 3 engine. Bất biến: LLM chỉ extract, code chấm điểm deterministic, agent chỉ CHỌN — không tự khai confidence vào công thức.
